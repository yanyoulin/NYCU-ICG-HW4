## How to Run Our code
### Requirements
- G++
- CMake
- VS Code with CMake Tools extension
### 方法
#### 直接下載此zip的src檔：  
配合 2025 NYCU ICG HW3，直接將此src檔取代作業的src，即可如HW3一樣執行
#### git clone：  
```bash
git clone --recursive https://github.com/yanyoulin/NYCU-ICG-HW4.git
```
用 VS Code 打開此資料夾後，若下載完存在 build 資料夾，請手動刪除  
設定完 Compiler 後直接執行即可  
VS Code should have CMake Tools extension already
### 操作
| 按鍵 | 效果 |
| :-: | --- |
| W | Zoom in |
| S | Zoom out |
| A | Rotate camera to the left |
| D | Rotate camera to the right |
| B | 海綿寶寶吹泡泡 |
| H | 投擲漢堡並爆炸 |
| C | 海綿寶寶身體波動 |
| Shift | Move camera down |
| Space | Move camera up |
